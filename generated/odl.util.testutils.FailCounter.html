FailCounter¶

class odl.util.testutils.FailCounter(test_name, err_msg=None, logger=<built-in function print>)[source]¶

Bases: object

Used to count the number of failures of something

Usage:

with FailCounter() as counter:
    # Do stuff

    counter.fail()

When done, it prints

*** FAILED 1 TEST CASE(S) ***

Methods

__eq__($self, value, /) Return self==value.
fail([string]) Add failure with reason as string.
