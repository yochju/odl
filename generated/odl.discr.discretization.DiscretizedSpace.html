DiscretizedSpace¶

class odl.discr.discretization.DiscretizedSpace(uspace, dspace, sampling=None, interpol=None)[source]¶

Bases: odl.discr.discretization.DiscretizedSet, odl.space.base_ntuples.FnBase

Abstract class for discretizations of linear vector spaces.

This variant of DiscretizedSet adds linear structure to all its members. The DiscretizedSet.uspace is a LinearSpace, the DiscretizedSet.dspace for the data representation is an implementation of , where is some Field, and both DiscretizedSet.sampling and DiscretizedSet.interpolation are linear Operator’s.
