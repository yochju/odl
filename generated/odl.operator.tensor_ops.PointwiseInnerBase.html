PointwiseInnerBase¶

class odl.operator.tensor_ops.PointwiseInnerBase(adjoint, vfspace, vecfield, weighting=None)[source]¶

Bases: odl.operator.tensor_ops.PointwiseTensorFieldOperator

Implemented to allow code reuse between the classes.
